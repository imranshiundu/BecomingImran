# Becoming Imran

A cinematic one-page personal site for Imran Shiundu, built as a restrained replica-inspired experience: oversized editorial typography, beige/black palette, sticky navigation, project rows, hover previews, proof cards, and a serious builder-focused about section.

## Direction

The site is intentionally small: one page, four anchors, no fake achievements, no bloated portfolio pages.

- **Home** — large editorial identity with one real portrait source.
- **Work** — selected projects presented like a filmography list.
- **Proof** — public record area for logs, writing, and serious updates.
- **About** — concise builder narrative and direct contact line.

## Files

- `index.html` — full page structure and content.
- `styles.css` — replica-inspired visual system and responsive UX.
- `script.js` — active navigation, work hover preview, contact button state, service worker registration.
- `assets/work/*.svg` — custom project cover artwork.
- `manifest.webmanifest` and `service-worker.js` — lightweight install/offline support.

## Asset note

The hero currently uses Imran's public GitHub avatar as the real portrait source. Replace it with a professional portrait later by changing the `img` source in `index.html` or by adding a local portrait asset.
